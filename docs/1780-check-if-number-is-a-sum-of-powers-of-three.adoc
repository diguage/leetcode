= 1780. Check if Number is a Sum of Powers of Three

https://leetcode.com/problems/check-if-number-is-a-sum-of-powers-of-three/[LeetCode - 1780. Check if Number is a Sum of Powers of Three ^]

Given an integer `n`, return `true` _if it is possible to represent _`n`_ as the sum of distinct powers of three._ Otherwise, return `false`.

An integer `y` is a power of three if there exists an integer `x` such that `y == 3^x^`.

 
<strong class="example">Example 1:*

[subs="verbatim,quotes"]
----
*Input:* n = 12
*Output:* true
*Explanation:* 12 = 3^1^ + 3^2^
----

<strong class="example">Example 2:*

[subs="verbatim,quotes"]
----
*Input:* n = 91
*Output:* true
*Explanation:* 91 = 3^0^ + 3^2^ + 3^4^
----

<strong class="example">Example 3:*

[subs="verbatim,quotes"]
----
*Input:* n = 21
*Output:* false
----

 
*Constraints:*


* `1 <= n <= 10^7^`
