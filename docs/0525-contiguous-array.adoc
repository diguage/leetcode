= 525. 连续数组

https://leetcode.cn/problems/contiguous-array/[LeetCode - 525. 连续数组^]

给定一个二进制数组 `nums` , 找到含有相同数量的 `0` 和 `1` 的最长连续子数组，并返回该子数组的长度。

*示例 1:*

....
输入: nums = [0,1]
输出: 2
说明: [0, 1] 是具有相同数量 0 和 1 的最长连续子数组。
....

*示例 2:*

....
输入: nums = [0,1,0]
输出: 2
说明: [0, 1] (或 [1, 0]) 是具有相同数量0和1的最长连续子数组。
....

*提示：*

* `1 \<= nums.length \<= 10^5^`
* `nums[i]` 不是 `0` 就是 `1`


== 思路分析

将 `0` 视为 `-1`，做前缀和计算，`0` 和 `1` 相等，则当前位置计算的数字必然在前面出现过。
