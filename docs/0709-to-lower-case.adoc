== 709. To Lower Case

https://leetcode.com/problems/to-lower-case/[LeetCode - To Lower Case]

Implement function ToLowerCase() that has a string parameter str, and returns the same string in lowercase.

 


*Example 1:*

[subs="verbatim,quotes,macros"]
----
*Input:* "Hello"
*Output:* "hello"
----


*Example 2:*

[subs="verbatim,quotes,macros"]
----
*Input:* "here"
*Output:* "here"
----


*Example 3:*

[subs="verbatim,quotes,macros"]
----
*Input:* "LOVELY"
*Output:* "lovely"
----
