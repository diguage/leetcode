= 128. 最长连续序列

https://leetcode.cn/problems/longest-consecutive-sequence/[LeetCode - 128. 最长连续序列^]

给定一个未排序的整数数组 `nums`，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 stem:[O(n)] 的算法解决此问题。

*示例 1：*

....
输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。
....

*示例 2：*

....
输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9
....

*提示：*

* `0 \<= nums.length \<= 10^5^`
* `-10^9^ \<= nums[i] \<= 10^9^`


== 思路分析

首先想到的思路是排序，然后再查找。

答案的思路是：+++<del>先将数字都添加到集合，然后遍历即可，只有没有前一位数字时（即为起点），开始统计个数。</del>+++上面的描述不够准确，应该这样说：使用两层循环，先使用外层循环寻找起点，在内循环中逐步判断是否有后续节点来查找更长连续数列。

求连续序列长度，可以先去重，再寻找当前序列起点，从起点向后查找，出现断点，则当前序列结束。取所有序列中，最长值即可。

看其他选手答案，使用并查集也可以解决。
