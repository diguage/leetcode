= 131. 分割回文串

https://leetcode.cn/problems/palindrome-partitioning/[LeetCode - 131. 分割回文串^]

给你一个字符串 `s`，请你将 `s` 分割成一些子串，使每个子串都是 *回文串*。返回 `s` 所有可能的分割方案。

*示例 1：*

....
输入：s = "aab"
输出：[["a","a","b"],["aa","b"]]
....

*示例 2：*

....
输入：s = "a"
输出：[["a"]]
....

*提示：*

* `+1 <= s.length <= 16+`
* `s` 仅由小写英文字母组成


== 思路分析

回溯：截取子串，判断是否为回文，如果是，则向前推进，截取下一个子串；否则，坐标向前移位。
