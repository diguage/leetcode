== 231. Power of Two

https://leetcode.com/problems/power-of-two/[LeetCode - Power of Two]

Given an integer, write a function to determine if it is a power of two.

*Example 1:*

[subs="verbatim,quotes"]
----
*Input:* 1
*Output:* true 
*Explanation: *2^0^ = 1

----

*Example 2:*

[subs="verbatim,quotes"]
----
*Input:* 16
*Output:* true
*Explanation: *2^4^ = 16
----

*Example 3:*

[subs="verbatim,quotes"]
----
*Input:* 218
*Output:* false
----
