= 504. 七进制数

https://leetcode.cn/problems/base-7/[LeetCode - 504. 七进制数^]

给定一个整数 `num`，将其转化为 *7 进制*，并以字符串形式输出。

*示例 1:*

....
输入: num = 100
输出: "202"
....

*示例 2:*

....
输入: num = -7
输出: "-10"
....

*提示：*

* `-10^7^ \<= num \<= 10^7^`
