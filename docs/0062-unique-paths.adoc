= 62. 不同路径

https://leetcode.cn/problems/unique-paths/[LeetCode - 62. 不同路径^]

一个机器人位于一个 `m x n` 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

*示例 1：*

....
输入：m = 3, n = 7
输出：28
....

*示例 2：*

....
输入：m = 3, n = 2
输出：3
解释：
从左上角开始，总共有 3 条路径可以到达右下角。
1. 向右 -> 向下 -> 向下
2. 向下 -> 向下 -> 向右
3. 向下 -> 向右 -> 向下
....

== 思路分析

常规解法就是动态规划；从数学的角度来看，也是一个组合问题。

[latexmath]
++++
C_{n}^{m} = \frac{A_{n}^{m}}{A_{m}^{m}} = \frac{n \cdot (n-1) \cdot \cdot \cdot(n-m+1)}{m!} = \frac{n!}{m! \cdot (n-m)!} = C_{n}^{n-m}
++++

对于本题而言，要想到达终点，需要往下走 latexmath:[n-1] 步，往右走 latexmath:[m-1] 步，总共需要走 latexmath:[n+m-2] 步。即：

[latexmath]
++++
C_{m+n-2}^{m-1} = \frac{(m+n-2)!}{(m-1)! \cdot (n-1)!}
++++
