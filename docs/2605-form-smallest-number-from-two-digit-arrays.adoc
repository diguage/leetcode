= 2605. Form Smallest Number From Two Digit Arrays

https://leetcode.com/problems/form-smallest-number-from-two-digit-arrays/[LeetCode - 2605. Form Smallest Number From Two Digit Arrays ^]

Given two arrays of *unique* digits `nums1` and `nums2`, return _the *smallest* number that contains *at least* one digit from each array_.
 
<strong class="example">Example 1:*

[subs="verbatim,quotes"]
----
*Input:* nums1 = [4,1,3], nums2 = [5,7]
*Output:* 15
*Explanation:* The number 15 contains the digit 1 from nums1 and the digit 5 from nums2. It can be proven that 15 is the smallest number we can have.
----

<strong class="example">Example 2:*

[subs="verbatim,quotes"]
----
*Input:* nums1 = [3,5,2,6], nums2 = [3,1,7]
*Output:* 3
*Explanation:* The number 3 contains the digit 3 which exists in both arrays.
----

 
*Constraints:*


* `1 <= nums1.length, nums2.length <= 9`
* `1 <= nums1[i], nums2[i] <= 9`
* All digits in each array are *unique*.
