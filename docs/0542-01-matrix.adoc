= 542. 01 矩阵

https://leetcode.cn/problems/01-matrix/[LeetCode - 542. 01 矩阵^]

给定一个由 `0` 和 `1` 组成的矩阵 `mat` ，请输出一个大小相同的矩阵，其中每一个格子是 `mat` 中对应位置元素到最近的 `0` 的距离。

两个相邻元素间的距离为 `1` 。

*示例 1：*

image::images/0542-01.png[{image_attr}]

....
输入：mat = [[0,0,0],[0,1,0],[0,0,0]]
输出：[[0,0,0],[0,1,0],[0,0,0]]
....

*示例 2：*

image::images/0542-02.png[{image_attr}]

....
输入：mat = [[0,0,0],[0,1,0],[1,1,1]]
输出：[[0,0,0],[0,1,0],[1,2,1]]
....

*提示：*

* `m == mat.length`
* `n == mat[i].length`
* `1 \<= m, n \<= 10^4^`
* `1 \<= m * n \<= 10^4^`
* `mat[i][j] is either 0 or 1.`
* `mat` 中至少有一个 `0`



== 思路分析

多源广度优先。以 `0` 为起点，向 `1` 广播，遇到 `1` 就停止，否则就继续向外广播。
