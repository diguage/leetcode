= 628. 三个数的最大乘积

https://leetcode.cn/problems/maximum-product-of-three-numbers/[LeetCode - 628. 三个数的最大乘积^]

给你一个整型数组 `nums`，在数组中找出由三个数组成的最大乘积，并输出这个乘积。

*示例 1：*

....
输入：nums = [1,2,3]
输出：6
....

*示例 2：*

....
输入：nums = [1,2,3,4]
输出：24
....

*示例 3：*

....
输入：nums = [-1,-2,-3]
输出：-6
....

*提示：*

* `3 \<= nums.length \<= 10^4^`
* `-1000 \<= nums[i] \<= 1000`


== 思路分析

最大的三个数相乘或者最大的正数和最小的两个负数相乘。
