= 148. Sort List

https://leetcode.com/problems/sort-list/[LeetCode - Sort List]

Sort a linked list in _O_(_n_ log _n_) time using constant space complexity.

*Example 1:*

[subs="verbatim,quotes,macros"]
----
*Input:* 4->2->1->3
*Output:* 1->2->3->4

----

*Example 2:*

[subs="verbatim,quotes,macros"]
----
*Input:* -1->5->3->4->0
*Output:* -1->0->3->4->5
----
